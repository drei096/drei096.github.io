Alchemical Breakout

Unity

Developed for my Mobile Applications Development class.

Alchemical Breakout is a 3D Sci-fi On Rails Shooter game heavily inspired by the House of the Dead arcade games. Instead of guns, it makes use of 4 elements (fire, water, electric, and ground) in which the player can use to cast magic against the elemental laboratory monsters. The game also features type effectivity similar to Pokemon in which certain elemental attacks will be effective/not effective against a certain monster (ex. a water attack will not be effective against a grass monster). Swiping the screen changes the player's element type, shaking the device allows the player to revive when he/she dies, and spreading allows the player to zoom in a bit into the game. This game was developed by myself and my class project partner.
